SISTEMA PRODIFY - GERENCIAMENTO DE PRODUTOS

Este sistema basicamente consiste em cadastro, atualização, exclusão e consulta de produtos. Na página "Cadastrar produto", é possível fazer o cadastro de um produto com as informações fornecidas. Na página "Lista de produtos", é possível listar todos os produtos cadastrados, consultar um por um através do número do ID ou do nome em um campo de busca, atualizar as informações do produto e excluir um produto já existente. Dentre as funcionalidades do sistema estão inclusas: colocar um produto em promoção e aplicar sobre ele um desconto de 0 a 100%, vincular uma imagem ao produto, calcular a média de preço dos produtos já cadastrados e calcular qual o preço do produto mais caro. De maneiras mais técnica, estão detalhadas abaixo as funcionalidades do Sistema PRODIFY:


Funcionalidades

- CRUD de produtos
- Cálculo de produto cadastrado com valor mais caro
- Cálculo da média de preço entre os produtos cadastrados
- Cálculo de desconto aplicado de 0 a 100% (não aceita decimais) sobre o preço do produto
- Campo de buscar produtos por ID e Nome
- Upload de arquivo de imagem para relacionar com o produto
- Swagger (Link acessível EM BREVE)
- Spring Security - JWT (Autenticação / Token / Role USER / Criptografia de senha / Expiração de token)
- Cadastro de usuários / Login / Verificação de login já cadastrado
- Relacionamento produto X usuário
- Contador de acessos por IP
- Certificado SSL - Endereço HTTPS
- Rodapá interativo com as últimas news
- Layout Modo Dark ON/OFF
