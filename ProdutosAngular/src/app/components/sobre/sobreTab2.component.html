SISTEMA PRODIFY - GERENCIAMENTO DE PRODUTOS

REPOSITÓRIO GITHUB - CÓDIGO DO PROJETO

Este sistema basicamente consiste em um gerenciamento de produtos, com fornecedores relacionados. Na página "Gerenciar produtos > Cadastrar produtos", é possível fazer o cadastro de um produto com as informações fornecidas, e relacionar um fornecedor, que por sua vez é cadastrado na página "Gerenciar fornecedores > Cadastrar fornecedores". Na página "Lista de produtos", é possível listar todos os produtos cadastrados, consultar um por um através do número do ID ou do nome em um campo de busca, atualizar as informações do produto e excluir um produto já existente. Dentre as funcionalidades do sistema estão inclusas: adicionar quantidade, colocar um produto em promoção e aplicar sobre ele um desconto de 0 a 100%, calcular frete, vincular uma imagem ao produto, calcular a média de preço dos produtos já cadastrados e calcular qual o preço do produto mais caro. De maneiras mais técnica, estão detalhadas abaixo as funcionalidades do Sistema PRODIFY:


Funcionalidades

- CRUD de produtos e fornecedores
- Relação Produto X Fornecedor
- Relação Produto X Usuário
- Relação Fornecedor X Usuário
- Calculo de quantidade X valor unitário
- Cálculo de produto cadastrado com valor mais caro
- Cálculo da média de preço entre os produtos cadastrados
- Cálculo de desconto aplicado de 0 a 100% (não aceita decimais) sobre o preço do produto
- Cálculo de frete
- Campo de buscar produtos por ID e Nome
- Upload de arquivo de imagem para relacionar com o produto
- Swagger com detalhes dos endpoints
- Spring Security - JWT (Autenticação / Token / Role USER / Criptografia de senha / Expiração de token)
- Cadastro de usuários / Login / Verificação de login já cadastrado
- Contador de acessos por IP
- Certificado SSL - Endereço HTTPS
- Totalmente responsivo
- Layout Modo Dark ON/OFF
- Integração com Google Cloud APIs para informações de localização
- Integração com Via CEP API para preenchimento automático e detalhado de endereço
- Integração com Brasil API para detecção de CNPJ e preenchimento automático e detalhado de informações da empresa
